# Flask User App

Simple Flask application for adding users.

## Features

- Add user with name and age
- Form validation
- Flash messages
- Simple Flask routing

## Project structure
app.py
templates/
static/

## Installation

Clone the repository:
git clone https://github.com/Draxienek/flask-user-app.git

Go to the project folder:
cd flask-user-app

Create virtual environment:
python -m venv venv


Activate venv (Windows):
venv\Scripts\activate

Install dependencies:
pip install -r requirements.txt

Run the app:
python app.py

## Run project on new machine

python -m venv venv
venv\Scripts\activate
